# Pixabay 新标签页 Chrome 扩展

一个简洁的 Chrome 新标签页扩展，使用 Pixabay 图片作为背景，支持随机、固定和收藏三种模式。

## 功能

- 每次打开新标签页时自动展示一张来自 Pixabay 的高质量图片。
- **随机模式**：随机从多个主题中挑选图片，点击刷新可换一张。
- **固定模式**：将喜欢的图片固定为常驻背景，不再自动更换。
- **收藏模式**：在随机模式下收藏的图片会加入收藏，收藏模式会随机展示这些图片。
- 图片信息底部展示摄影师与标签，并提供链接跳转至 Pixabay 原图页面。

## 安装方式

1. 打开 Chrome 浏览器，在地址栏输入 `chrome://extensions/` 并回车。
2. 打开右上角「开发者模式」。
3. 点击「加载已解压的扩展程序」。
4. 选择本项目所在的 `pixabay-newtab` 文件夹。
5. 打开新标签页即可看到效果。

## 文件说明

| 文件 | 说明 |
|------|------|
| `manifest.json` | 扩展配置文件，声明新标签页覆盖页与权限 |
| `newtab.html` | 新标签页 HTML |
| `newtab.css` | 新标签页样式 |
| `newtab.js` | 核心逻辑：调用 Pixabay API、模式切换、收藏与固定 |

## 使用提示

- 首次加载可能需要联网获取图片，请保持网络畅通。
- 固定和收藏的图片会保存在浏览器的本地存储中，仅在当前设备生效。
- 如需更换 API key，请修改 `newtab.js` 顶部的 `API_KEY` 常量。
